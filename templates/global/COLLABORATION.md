# 协作协议模板

> 版本: v3.0（双角色模型）

## 协作模型

```
你（审查者 / 决策者）
    │
    ├── 提需求（方向性描述即可）
    ├── 审查重要变更（架构 / 契约 / 高风险）
    └── 裁决争议
    │
    ▼
我（执行 Agent）
    │
    ├── 共享数据契约
    ├── .clinerules（硬约束）
    └── memory/（规则沉淀）
```

## 审查分层

| 层级 | 范围 | 审查方式 | 你的投入 |
|------|------|---------|---------|
| Tier 1 | 日常修复、配置、文档 | 自验 + 测试全绿 | 看一眼 diff |
| Tier 2 | 架构调整、契约变更、API 重写 | 你审 diff + PRD | ~5 分钟 |
| Tier 3 | DB schema、安全凭证、回滚敏感 | 你必须在场 | 一起 review |

## Review Checklist

```
□ 安全检查：密钥/env 硬编码？
□ 架构合规：共享契约被绕过？
□ 回归测试：bug fix 带测试了？
□ 错误处理：异常正确处理？
□ 边界：空列表、None、超大输入？
□ 文档同步：PRD/CHANGELOG 更新了？
□ 测试全绿？
```
