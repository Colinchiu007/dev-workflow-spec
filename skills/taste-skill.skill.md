# Taste Skill — 反模板前端设计合集

> 完整内容已安装为 Cowork skill（6 个子技能合集）。

## 包含的子技能

| 技能 | 适用场景 |
|------|---------|
| 通用反模板设计 | 任何需要设计感的页面 |
| GPT 严格模式 | 需要精确遵循设计规范 |
| 图片转代码 | 从设计稿截图到代码 |
| 重设计 | 升级现有 UI |
| 柔和极简主义 | 温暖、编辑式的视觉风格 |
| 工业 brutalism | 大胆、粗野的视觉风格 |

## 在项目中的使用

在 `templates/project/AGENTS.md` 的阶段 5（编码）中引用 taste-skill：
- 需要设计的新功能 → 激活 taste-skill
- UI 修改 → 参考 taste-skill 的子技能指南
